Паси та Ланцюги

Якісні приводні паси та ланцюги для всіх типів обладнання

← Повернутися на головну

Паси клинові класичні

Стандартні клинові паси для передачі обертального моменту в промисловому обладнанні.

Паси клинові вузькопрофільні

Компактні вузькопрофільні паси для обмеженого простору та високих швидкостей.

Паси багатоклинові

Багаторебристі паси для передачі великих потужностей та високих швидкостей обертання.

Паси поліклинові

Сучасні поліклинові паси з підвищеною гнучкістю та довговічністю.

Паси зубчасті

Синхронні зубчасті паси для точної передачі без проковзування.

Паси шестигранні (двохсторонні)

Спеціальні шестигранні паси для складних приводних систем з двостороннім використанням.

Паси варіаторні

Паси для варіаторних передач з можливістю плавної зміни передавального відношення.

Паси плоскі

Традиційні плоскі паси для легких навантажень та великих швидкостей.

Ланцюги приводні роликові

Стандартні роликові ланцюги для промислового застосування з високою надійністю.

Ланцюги для с/г техніки

Спеціалізовані ланцюги для сільськогосподарської техніки, стійкі до забруднень.

Ланцюги зі спецкріпленням

Ланцюги з особливими типами кріплень для специфічних застосувань.

Ланки з'єднувальні та перехідні

Замкові ланки, з'єднувачі та перехідні елементи для монтажу та ремонту ланцюгів.
